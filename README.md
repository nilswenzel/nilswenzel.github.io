# nilswenzel.github.io

## Local development
1. Start devcontainer

2. Install dependencies
``` shell
npm install
```

3. Start dev server
```shell
npm start
```

## Deploy to Github Pages: [nilswenzel.github.io](https://nilswenzel.github.io/)
```shell
npm run deploy -- -m "<message>"
```
